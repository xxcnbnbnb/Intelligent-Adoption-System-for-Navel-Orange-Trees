# 脐橙树智慧认养系统

## 项目介绍

脐橙树智慧认养系统是一个基于现代信息技术的农业认养平台，旨在通过数字化手段实现脐橙树从种植、管理到收获、销售的全流程管理。系统为消费者提供了一种全新的农业体验模式，允许用户认养脐橙树，实时查看树木生长状态，参与树木管理过程，最终获得自己认养树木的果实。

## 技术栈

### 前端技术
- **框架**：Vue 3
- **状态管理**：Pinia
- **路由**：Vue Router
- **UI组件库**：Element Plus
- **HTTP客户端**：Axios
- **构建工具**：Vite

### 后端技术
- **运行环境**：Node.js
- **Web框架**：Express
- **ORM**：Knex.js
- **认证**：JWT
- **数据库**：MySQL
- **安全**：Helmet、CORS、Rate Limit

## 系统架构

系统采用前后端分离的架构设计：

### 前端架构
- **用户端**：提供用户注册登录、树木浏览、认养管理、生长记录查看、物流跟踪、互动等功能
- **管理员端**：提供系统管理、用户管理、树木管理、认养订单管理、收获管理、操作日志等功能

### 后端架构
- **API层**：提供RESTful API接口
- **服务层**：处理业务逻辑
- **仓库层**：数据访问
- **模型层**：数据模型定义
- **中间件**：认证、日志、错误处理等

## 安装部署

### 后端部署

1. **安装依赖**
   ```bash
   cd OrangeTree
   npm install
   ```

2. **配置环境变量**
   - 复制 `.env.example` 文件为 `.env`
   - 编辑 `.env` 文件，配置数据库连接信息和其他环境变量

3. **启动服务器**
   ```bash
   npm start
   ```
   服务器默认运行在 `http://localhost:3000`

### 前端部署

1. **安装依赖**
   ```bash
   cd OrangeTreeVue3
   npm install
   ```

2. **配置环境变量**
   - 复制 `.env.development` 文件为 `.env.local`
   - 编辑 `.env.local` 文件，配置API基础URL

3. **开发模式运行**
   ```bash
   npm run dev
   ```
   前端默认运行在 `http://localhost:5173`

4. **构建生产版本**
   ```bash
   npm run build
   ```
   构建产物位于 `dist` 目录

## 功能模块

### 用户端功能

1. **用户管理**
   - 用户注册、登录
   - 个人信息管理

2. **树木浏览**
   - 浏览可认养的脐橙树列表
   - 查看树木详细信息（位置、树龄、产量等）

3. **认养管理**
   - 在线认养脐橙树
   - 生成认养订单
   - 支付认养费用

4. **生长记录**
   - 查看认养树木的生长状态
   - 浏览生长照片和视频
   - 查看生长数据统计

5. **物流管理**
   - 查看果实收获后的物流信息
   - 跟踪包裹状态

6. **互动功能**
   - 与管理员进行互动
   - 提出问题或建议

### 管理员端功能

1. **仪表盘**
   - 系统概览
   - 关键数据统计

2. **用户管理**
   - 管理注册用户信息
   - 查看用户操作记录

3. **树木管理**
   - 添加、编辑、删除脐橙树信息
   - 更新树木状态

4. **认养订单管理**
   - 处理用户认养订单
   - 确认认养信息

5. **收获管理**
   - 管理果实收获计划
   - 记录收获数据

6. **操作日志**
   - 记录管理员操作历史
   - 系统运行日志

## 系统特点

1. **数字化管理**：实现脐橙树全生命周期的数字化管理，提高管理效率

2. **实时监控**：通过物联网设备实时监控树木生长状态，及时发现问题

3. **透明化**：消费者可以实时查看树木生长过程，增强信任度
